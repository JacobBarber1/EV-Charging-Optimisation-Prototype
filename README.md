
# EcoCharge

The project is deployed on Render and can be viewed [here](https://ecocharge-prototype.onrender.com/).

## About the project

The application provides users with a path to the nearest charging station and generates an optimal charging schedule based on userâ€™s specified times and fluctuating electricity prices. This unique feature addresses a real-world problem faced by EV users and demonstrates the potential of computational solutions in improving the efficiency and cost-effectiveness of EV charging

## Installation

#### Dependencies:

- [Chart.js](https://www.chartjs.org/)
- [Leaflet](https://leafletjs.com/)

#### To install and setup the project with the required dependencies:

- Clone the repository.
- Navigate into the project directory and create a client folder by running `npm create vite@latest`, then initialise it using `npm i`.
